# kamditis.github.io

Portfolio site for Kevin Amditis's plugins and tools.

## Live Site

https://kamditis.github.io

## Adding a New Plugin

1. Open `index.html`
2. Copy an existing `<div class="plugin-card">` block
3. Update the icon, name, platform, description, and links
4. Commit and push
