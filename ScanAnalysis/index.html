ScanAnalysis

Point Cloud Deviation Analysis for Autodesk Revit

Download ScanAnalysis

Supports Revit 2023, 2024, 2025, 2026 | ~4 MB

Installation

  1. Extract the downloaded ZIP file
  2. Right-click Install-ScanAnalysis.ps1 and select Run with PowerShell
  3. Select your Revit version(s) when prompted
  4. Restart Revit
  5. Look for the Scan Analysis tab in the ribbon

Requirements

  • Autodesk Revit 2023, 2024, 2025, or 2026
  • CloudCompare - Download here (for deviation analysis)
  • WebView2 Runtime - Installed automatically (for 360 panorama viewer)

Uninstall

Run the installer with the uninstall flag:

.\Install-ScanAnalysis.ps1 -Uninstall
