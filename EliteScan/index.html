EliteScan

Point Cloud Deviation Analysis for Autodesk Revit

Version 1.1.0
Download Installer

Windows Installer (~4.5 MB) | Supports Revit 2022-2026

Deviation Analysis

Compare Revit model geometry against point cloud scans with per-element accuracy scoring and grade assignments (A-F).

360° Panorama Viewer

View equirectangular panorama images from scan locations with hotspot navigation and Revit view synchronization.

QA Verification

Mark elements as verified with notes and screenshots. Generate professional HTML reports for client deliverables.

Smart Scan Placement

Automatically place scan location families at RCS scan origins with correct positioning and metadata.

Multi-Scan Aggregation

Elements captured by multiple scans are aggregated with conflict detection and selectable scan sources.

Quick Overlays

Apply color-coded deviation visualization to elements in Revit views based on their accuracy scores.

System Requirements

  • Autodesk Revit 2022, 2023, 2024, 2025, or 2026
  • Windows 10/11 (64-bit)
  • WebView2 Runtime (for 360° Viewer)
  • CloudCompare (for deviation analysis)
  • Point cloud scans with panorama images
