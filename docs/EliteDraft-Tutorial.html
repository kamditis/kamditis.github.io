EliteDraft

A tutorial walkthrough of every ribbon button, dialog, and workflow in the MEP fabrication plugin

v1.0.15 / Tutorial & Reference Guide

Ribbon overview

The plugin adds an "Elite Draft" tab to the Revit ribbon with three panels:

Hangers panel

  • Hanger Settings — Configure placement rules
  • Single Pipes — Place individual hangers
  • Pipe Rack — Place trapeze hangers
  • Duct Hangers — Place duct hangers
  • Rack Adjust — Adjust trapeze positions
  • Fill Info — Fill hanger parameters

Annotation panel

  • Quick Tag — Tag duct fittings
  • Tag Settings — Configure tag positions
  • Elbow Extension — Fill elbow params
  • Transition — Fill transition params
  • Batch Spools — Spool sheet manager
  • Renumber — Renumber spool parts

View panel

  • Isolate Service — Isolate by service name
  • Hide Service — Hide by service name

Supports Revit 2022-2026. Revit 2022-2024 use .NET Framework 4.8; Revit 2025-2026 use .NET 8.0. Icons auto-swap between Light and Dark themes.

Section 01

Hangers panel

Automated hanger and trapeze placement for fabrication pipes and ducts

Hanger ribbon buttons

Hangers
Button What it does
Hanger Settings Opens the main Hanger & Support Layout dialog. Configure placement rules, hanger button assignments, material categories, and spacing standards. Also acts as the launch point for running hanger placement.
Single Pipes Places individual hangers on selected (or all visible) fabrication pipes. Uses the spacing rules and hanger button defined in Hanger Settings. Runs silently without opening a dialog.
Pipe Rack Places trapeze (rack) hangers on grouped pipes. Pipes at similar elevations and running parallel are grouped together. Uses min/max pipe counts and rod setback from settings.
Duct Hangers Places duct hangers on selected (or all visible) fabrication ducts. Supports both round and rectangular duct types with separate hanger button assignments.
Rack Adjust Adjusts positions of existing trapeze/rack hangers. Use after placing trapezes to fine-tune rod positions and unistrut lengths.
Fill Info Fills shared hanger parameters across all hangers in the project: Elevation Offset, Rod Length, Rod Length #2, Total Length, Unistrut Length, and Free Size. Reads values from fabrication dimensions.

Hanger Settings dialog — Placement Settings

Hanger Settings

The top section of the Hanger & Support Layout dialog configures global placement behavior and hanger button assignments.

Placement settings

  • Place at pipe ends — Checkbox to always place hangers at pipe start/end points
  • Clash detection — Checks hanger positions against nearby MEP elements to avoid rod and strap clashes
  • Edge clearance — Minimum distance from pipe ends (ft & in)
  • Ignore short pipes — Skip pipes shorter than a threshold (in inches)

Trapeze settings

  • Min pipes for trapeze — Minimum number of parallel pipes to group
  • Max pipes per trapeze — Maximum pipes in a single trapeze
  • Rod setback — Distance from outside of insulation to rod centerline (inches)
  • Parallel angle tolerance — Maximum angle between pipes to consider them parallel (degrees)
  • Grouping vertical tolerance — Maximum vertical distance between pipes to group them

Hanger selection (4 ComboBoxes with previews)

  • Single Pipe — Default hanger button for individual pipe hangers. Preview thumbnail shows the selected hanger type.
  • Pipe Rack — Hanger button for trapeze/rack configurations
  • Round Duct — Hanger button for round duct hangers
  • Rect Duct — Hanger button for rectangular duct hangers
EliteDraft Tutorial 5

Hanger Settings dialog — Material Categories & Spacing

Hanger Settings

The bottom section defines spacing rules per material category. Each category has a list of pipe size ranges with maximum spacing distances.

Category list (left)

  • Material + Medium categories with enable/disable checkboxes
  • Add — Create a new material category
  • Remove — Delete the selected category
  • Each category can have its own "Require end supports" override

Size rules DataGrid (right)

  • Max NPS (in) — Upper limit of pipe size range
  • Max Spacing (ft) — Maximum distance between hangers
  • Hanger Button — Optional per-size override (ComboBox)
  • Enabled — Toggle individual size rules on/off
ButtonAction
Load ETB Standards Replace all spacing values with Engineering Toolbox standards (MSS SP-58, ASME B31.1) based on the category's material and medium.
Add Standard Sizes Add missing standard pipe sizes (1/2" - 18") while keeping existing spacing values intact.
Reset Category Reset the selected category to generic default spacing values.

Footer buttons: Save Settings... and Load Settings... export/import the full configuration as JSON. Cancel discards changes. Finish saves and closes.

Annotation panel — Tagging

Quick Tag, Duct Annotation Settings, and parameter fill commands for elbows and transitions

Quick Tag dialog

Annotation

The Quick Tag dialog provides a fast way to tag fabrication duct fittings in the active view. Select which fitting types to include, then tag by selection or by visibility.

Fitting type checkboxes

  • Straights — Straight duct sections
  • Radius Elbows — Radius Bend, Reducing Bend, Drop Cheek Bend
  • Square Elbows — Square Bend, Elbow
  • Transitions — Transition, Radius Offset, Mitred Offset, Square to Round

Dimmed checkboxes indicate no tag positions are configured for that type. Open Tag Settings to configure.

Actions

  • Tag Selected — Tags only the currently selected duct fittings in the view
  • Tag All Visible — Tags every visible duct fitting in the active view
  • Progress bar — Displays tagging progress during batch operations
  • Settings... — Opens the Duct Annotation Settings dialog

Ducts facing up/down relative to the view (appearing as circles in plan) are automatically skipped.

EliteDraft Tutorial 8

Duct Annotation Settings dialog

Annotation

Configure tag positions per fitting type using an interactive 3x3 grid diagram. Each position can have its own tag family, offsets, orientation, and leader settings.

Fitting type tabs

  • Straight — Rectangular duct body with C1/C2 connectors
  • Radius Elbow — 90-degree bend with inner/outer sides
  • Square Elbow — 90-degree mitered bend
  • Transition — Reducer with large (C1) and small (C2) ends

Each tab shows a schematic diagram with 9 clickable position nodes arranged in a 3x3 grid. Amber-highlighted nodes have tags assigned.

Position settings (right panel)

  • Enabled — Toggle this position on/off
  • Tag Family — Select which tag family to place at this position
  • Perpendicular offset — Distance from duct surface (+ away from duct)
  • Parallel offset — Distance along flow direction (+ downstream)
  • Orientation — Along Duct, Horizontal, or Vertical
  • Show Leader — Enable/disable leader line
  • Clear Position — Remove tag assignment from this node

EliteDraft Tutorial 9

Elbow Extension & Transition parameter commands

Annotation

These silent commands read fabrication dimensions from parts in the active view and write calculated values to shared parameters for use in tags and schedules.

Elbow Extension

  • Scans all fabrication parts in the active view
  • Reads Top Extension, Bottom Extension, and Inner Radius from fabrication dimensions
  • Writes values to shared parameters: SM - Top Extension SM - Bottom Extension SM - Elbow Radius
  • Skips parts without elbow dimensions (straights, transitions, etc.)

Transition

  • Identifies transitions by Fabrication Fitting Description (Transition, Radius Offset, Mitred Offset, Square to Round)
  • Calculates vertical offset type: FT FB O/C BU/BD
  • Writes to shared parameters: SM - OFFSET (text), SM - SET (text), SM - CALC TRANSITION LENGTH (number)
  • Handles rotated parts and vertical flow using connector world-space positions

Spool Manager — Configuration Tabs

Batch Spools

The center column contains three configuration tabs that control what gets generated on each spool sheet.

General Settings tab

  • Title Block — ComboBox to select the sheet title block family
  • View Template — Template applied to all generated views
  • Schedule Template — Template for the BOM schedule
  • Tag Family — Tag family used for item number tags on the ISO view
  • Sheet Parameters — Sheet Function (editable), Sheet Number (from spool name), Area/Discipline (from part level), Weight/Welds (auto-calculated), Spool Note (set in right panel)
  • BOM Text Types — Title, Header, and Data text note types for the BOM table formatting
  • Spool Map — Map name applied to the sheet

Views & Schedules tab

  • 3D Ortho (ISO) — 3D orthographic view
  • Plan View — Top-down plan view
  • Front Elevation — Front section view
  • Side Elevation — Side section view
  • Bill of Materials — BOM schedule

Sheet Layout tab

  • Interactive canvas with title block linework
  • Draggable zones: ISO, PLAN, FRONT, SIDE, BOM
  • Drag edges/corners to resize zones
  • Mouse wheel to zoom (0.5x-4x)
  • Middle-click drag to pan
  • Zone properties panel: X, Y, W, H in inches
  • Zone size controls the view scale on the sheet

EliteDraft Tutorial 14

Renumber command

Annotation

The Renumber button renumbers fabrication parts within selected assemblies (spools). It runs silently on the current selection.

How it works

  • Select one or more assemblies (spools) in the Revit model
  • Click Renumber on the ribbon
  • Parts are grouped by description, size, and centerline length
  • Like items receive the same item number; unique items get sequential numbers
  • Also updates spool weight and weld inch calculations on the spool sheet
  • Clears all existing item numbers first, then reassigns from scratch

Tip: Run Renumber after modifying assembly contents (adding/removing parts) or after generating spool sheets to ensure item numbers and calculations are up to date.

View panel

Service-based isolation and hiding for fabrication parts

Isolate Service & Hide Service

View

These two commands filter the active view based on the fabrication service name. They operate on the current selection or prompt for a pick if nothing is selected.

Isolate Service

  • If no elements selected: prompts you to pick a fabrication part, then isolates all parts with the same service name
  • If elements are selected: reads service names from all selected parts and isolates every matching element
  • Uses temporary isolation in the active view (reversible via Revit's "Reset Temporary Hide/Isolate")
  • Great for focusing on a single trade's ductwork or piping

Hide Service

  • Inverse of Isolate: hides all fabrication parts matching the selected service name(s)
  • Same selection behavior: pick one element or use current selection
  • Uses temporary hide in the active view
  • Useful for removing a noisy service to see other trades more clearly

Both commands read the FABRICATION_SERVICE_NAME built-in parameter. They only affect fabrication parts (FabricationPart class) visible in the current view.
